• P4.13 Write a program that prints all powers of 2 from 2^0 up to 2^20.
-------------------------------------------------------------------------------------------------------------------------------------
public class hello {
	public static void main(String [] args)
	{
		for(double i=0;i<21;i++)
		{
		System.out.printf("2^%1.0f=%1.0f",i,Math.pow(2, i))	;
		System.out.println("");
		}
	}
}
